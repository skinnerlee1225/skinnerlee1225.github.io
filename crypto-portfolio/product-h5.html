product overview

產品定位與設計挑戰

這是搭配 DeFi 理財平台的 H5 拉新活動。三大活動同時進行，但每個活動目標不同：社群報銷瞄準有線下號召力的 KOL、VIP 挑戰賽瞄準有大客戶基礎的核心節點、每日排行榜瞄準競爭意識強的活躍用戶。三個活動互不干擾、共用同一個業績計算規則。

關鍵決策 #1
三活動分流 vs 單一大活動

同時做三個活動而非一個大活動。原因：不同類型用戶需要不同的激勵 — KOL 要報銷、核心節點要 VIP 衝級、活躍用戶要每日榜單。

關鍵決策 #2
排行榜每 5 分鐘刷新

不做即時刷新（成本高），也不每小時刷新（沒緊張感）。5 分鐘是兼顧用戶體驗與後端壓力的平衡點。

關鍵決策 #3
VIP 挑戰分 3 期釋放

40% / 30% / 30% 分 3 期，避免用戶達標後立即贖回離場。釋放期間業績下滑 ≥30% 暫停、≥50% 追回，鎖定長期承諾。

關鍵決策 #4
社群報銷需人工審核

不像其他活動自動發獎，社群報銷強制人工審核（1~3 工作天）。原因：要驗證活動真實性、防 AI 生成素材作弊。

campaign rules

三大活動規則

2.1活動一：社群報銷升級版

達標條件（全部滿足）：

  • 新增團隊業績 ≥ 5,000 USDT
  • 新增有效用戶 ≥ 10 人
  • 舉辦線下/社群活動（需上傳素材）

素材要求：活動影片（必須，≤200MB）+ 活動照片 ≥3 張（jpg/png，每張 ≤10MB）+ 品牌文宣露出截圖

獎勵：500 USDT，需人工審核（1~3 工作天），審核通過後 T+1 自動入帳。每期限申請 1 次，審核不通過可補件 1 次。

2.2活動二：VIP 級別挑戰賽

規則：當前等級 + 3 級 = 挑戰目標。當前需 ≥VIP1 才能參賽。

起始目標時限獎勵
VIP1VIP430 天5,000 USDT
VIP2VIP545 天10,000 USDT
VIP3VIP660 天30,000 USDT
VIP4VIP790 天80,000 USDT
VIP5VIP8120 天200,000 USDT

獎勵釋放：達成目標等級時立即釋放 40%，30 天後釋放 30%（業績不得下滑 ≥30%），60 天後釋放最後 30%。下滑 ≥50% 即追回已發。

2.3活動三：每日排行榜

業績計算：當日新增投資 - 當日撤資 - 當日提現。存續時間 ≥24 小時的資金才計入。團隊內互轉、自充自投不計入。

統計時間：UTC+8 00:00:00 ~ 23:59:59，快照時間 23:59:59 鎖定。每 5 分鐘計算一次。最低門檻 ≥1,000 USDT 才能上榜。

anti-fraud

風控與防作弊

4.1防自刷機制

維度規則觸發動作
設備指紋同一 device_fingerprint 註冊多帳號僅保留首帳號，其餘不計入新增
IP 地址同 IP 24h 內註冊 ≥ 3 帳號標記異常，人工審核
KYC同一身份證/護照號碼僅首帳號有效，重複註冊拒絕
行為模式批量註冊時間間隔 < 30 秒自動標記為機器人，全部不計入

4.2有效用戶定義

  • 註冊 + 通過 KYC + 首次有效投資 ≥ 100 USDT
  • 首筆投資需存續 ≥ 7 天，否則不計入
  • 註冊後 14 天內無任何操作，自動標記為無效
  • 系統偵測同設備/同IP/同KYC 的關聯帳號

4.3排行榜防刷

規則觸發條件處理方式
異常波動單日業績突增 ≥ 500%自動凍結排名，人工覆核
短時衝量1 小時內集中大額入金標記觀察，延遲計入
集中提現上榜後 48h 內提現 ≥ 80%追回當期獎勵
多帳號協作多帳號向同一上級集中投資合併計算，異常帳號凍結

4.4報銷審核規則

審核項目合格標準不合格處理
影片真實性可辨識活動場地、參與者、時間退回 + 允許補件 1 次
照片數量≥ 3 張不同角度退回 + 允許補件
品牌露出可見品牌 LOGO / 名稱退回 + 允許補件
參與人數照片/影片可辨識 ≥ 5 名參與者退回
素材時效活動日期需在活動期間內不合格，不可補件
素材原創AI 偵測 + 人工比對永久取消資格

4.5異常處理分級

Level 1 輕微：單項異常指標 → 標記觀察，不影響正常使用。

Level 2 中度：多項異常或單項嚴重 → 凍結相關活動獎勵，發送人工覆核工單。

Level 3 嚴重：確認作弊或利用漏洞 → 取消活動資格 + 追回已發獎勵 + 帳號降級。

Level 4 封號：重複嚴重違規或涉及金融詐騙 → 永久封禁帳號 + 凍結所有資金。

4.6技術實作

前端：
- React / Vue3 + Vite
- TailwindCSS
- 排行榜：Polling (5min interval)
- 倒數計時：Server Time Sync 避免客戶端時間篡改

後端：
- RESTful API + Rate Limiting
- 排行榜計算：定時任務每 5 分鐘跑一次 SQL 聚合
- 快照機制：23:59:59 鎖定排名寫入 daily_ranking
- 風控引擎：獨立微服務，事件流即時判斷
- 獎勵發放：排程任務 + 異步處理 + 冪等保護
